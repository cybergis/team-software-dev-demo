Team Software Development Guide
===============================


The `Github repository can be found here <https://github.com/cybergis/team-software-dev-demo>`_.

**Author:** `Alexander Michels <https://alexandermichels.github.io/>`_


This site has the goal of walking through the basics of team software development, covering some key tools and concepts to do this work efficiently. The example code here and information on how to deploy a Github pages website are from a previous tutorial `on creating Github Pages for Software <https://cybergis.github.io/github-pages-demo/>`_.
